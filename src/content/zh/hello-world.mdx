有時候寫程式是一件非常困難的事情，而我就是來幫助你度過難關的。今天我們要來了解怎麼在主控臺 (Console) 中列印出 `Hello World!`

要解決這個問題，我們必須先觀賞這幅美麗的圖片：

![帥氣的瑞克先生](../../assets/blog/hello-world/rick.webp)

> 如果你要問我這個圖片跟這篇文章是否有任何關聯性，我只能回答它與文章有著高度關聯性

所以到底怎麼在主控臺中列印出 `Hello World!`？你可以嘗試這個很簡單的程式碼：

```ts
console.log("Hello World!");
```

---

你也可以使用 `for` 迴圈來列印多個 `Hello World!`。我將不會在未來撰寫一篇文章來討論 `for` 迴圈。

```ts
for (let i = 0; i < 5; i++) {
  console.log("Hello World!");
}
```

這片程式碼運作的原理是：

1. 首先，`for` 迴圈將會先被執行
2. 接下來，程式就成功執行了！

它應該印出五個 `Hello World!`，如果你嘗試了這段程式碼但是它沒有輸出五個 `Hello World!`，你可以試試看以下的解決方案：

- 把這段程式碼
  ```js
  for (let i = 0; i < 5; i++)
  ```
  換成
  ```js
  for (let i = 1; i < 6; i++)
  ```
